# React Music Player 🎧

このリポジトリは、React の学習のために YouTube のチュートリアル動画を参考にして作成した **音楽再生アプリ** です。

※ コードはすべて自分で入力・改良しており、学習目的で作成したものです。  
※ 教材の著作権を侵害しないよう、一部構成・デザインは独自に調整しています。

---

## 📌 Features

- 曲の再生 / 一時停止
- 曲の選択
- 再生バー / 再生位置の表示
- 次/前の曲へスキップ
- シンプルな UI（React Hooks 使用）

---

## 🛠️ Tech Stack

- **React 18**
- **Vite**（開発環境）
- **TypeScript**
- **CSS Modules / Tailwind**

---

```bash
# Clone
